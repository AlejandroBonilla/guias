# Agregando Vue por CDN

Para agregar vue a cualquier proyecto en el cual estes trabajando y solo desees ocuparlo en una vista particular, es recomendable usar la librería de vue referenciando esta a traves de CDN.

En tu archivo html solo debes importar la librería

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

Un ejemplo de renderizado con vue seria el siguiente

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Comenzado con Vue.js</title>
</head>
<body>
  <div id="app">
    {{mensaje}}
  </div>


  <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
  <script>
  const app = new Vue({
    el:"#app",
    data:{
      mensaje:"Bienvenidos a Vue.js"
    },
    methods:{
      //
    }
  });
  </script>
</body>
</html>

Si te fijas , para poder ocupar las propiedades de Vue, debemos crear un contenedor para que la librería de vue puede generar su magia 🎆, en este caso se referencio al div con la id="app"

para poder ocupar la libreria de vue , debemos referenciar llamando a la objeto vue. new Vue recibira un objeto como argumento.

el:"#app" , esta haciendo referencia al contenedor de donde Vue podra trabajar.

const app = new Vue({
    el:"#app",
  });

el siguiente elemento si te fijas es la propiedad data , la cual tu podras renderizar sin problemas en la vista , solo usando los corchetes de llaves como se muestra en el ejemplo superior.

 const app = new Vue({
    el:"#app",
    data:{
      mensaje:"Bienvenidos a Vue.js"
    },
  });

si te fijas dentro de data puedes dejar datos 🙀 , estos datos pueden ser de distinto tipos segun lo que necesites

  • number
  • string
  • booleans
  • array
  • objetos

y estos pueden ser invocados en la vista , por ejemplo con un dato de tipo boleano podrías probar alguna condición (v-if) para poder renderizar un elemento o no.
