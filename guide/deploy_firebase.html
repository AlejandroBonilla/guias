# Firebase Deploy

# Empaquetar

Una vez terminada nuestra app vamos a realizar el build para llevarla a producción.

npm run build

# Instalar firebase-tools

debemos instalar herramientas de firebase ,esto lo haremos una sola vez , ya que la instalación sera global.

npm install -g firebase-tools

# Login firebase

una vez instalado , ahora ya podremos utilizarlo.

si ejecutamos

firebase login

podremos logearnos en firebase para hacer las operaciones de deploy posteriores

# Init firebase

una vez logueados debemos inicializar nuestro proyecto para con firebase

firebase init

al correr este comando , firebase nos preguntara algunas configuraciones, para inicializar.

# hosting firebase

Seleccionaremos la opción Hosting: Configure files for Firebase Hosting and (optionally) set up GitHub Action deploys y daremos enter.

# hosting setup

# Carpeta publica dist

En la siguiente seccion debemos indicar a firebase cual sera nuestra carpeta publica , indicaremos dist , la carpeta anteriormente generada por npm run build

# single app index

En esta sección debemos indicar que queremos configurar una single page aplication

# No automatic builds

Ya que no es tema del curso no seleccionaremos el deploy automatico con github

# NO OverWrite

Ya que nuestra aplicación ya fue construida en la carpeta dist esta el index.html de nuestra aplicación, si permitimos que firebase la reescriba perderemos el enlace a nuestra aplicación.

# init success

Si todo va bien debemos obtener el siguiente mensaje de salida

# deploy

Luego de hacer la configuración inicial de firebase podremos hacer deploy de nuestra aplicación

firebase deploy

Si todo sale correcto firebase nos entregar la url donde esta servida nuestra aplicación 🙉.
