# Dependencias en express

# Introducción

Si quisiéramos usar algunas dependencias dentro de nuestro proyecto en express , en algunos casos podríamos ocupar cdn , pero si quisieramos instalar la dependencia en nuestro proyecto podríamos ocupar

npm install nombre_dependencia

# Axios

Para instalar axios debes ir a la raíz de tu proyecto y abrir el terminal y colocar la siguiente instrucción

  npm install axios

luego lo que debemos hacer es ir al archivo app.js (esta en la raíz del proyecto) , luego debemos agregar las siguientes lineas

app.use('/assets', [
  express.static(__dirname + '/node_modules/axios/dist'),
]);

una vez modificado la vista , debes bajar el servidor y levantarlo nuevamente para que tome los cambios hechos en app.js , para bajarlo debemos presionar las letras ctl+c

Si lo quisieras ocupar dentro de alguna vista ejs lo que debes hacer es solo invocar al script por ejemplo

<script src="/assets/axios.js"></script>
<script src="/actions/my_script.js"></script>

# Bootstrap

Para instalar bootstrap debes ir a la raíz de tu proyecto y abrir el terminal y colocar la siguiente instrucción

  npm install bootstrap

luego lo que debemos hacer es ir al archivo app.js (esta en la raíz del proyecto) , luego debemos agregar las siguientes lineas

app.use('/assets', [
  express.static(__dirname + '/node_modules/bootstrap/dist/js/'),
  express.static(__dirname + '/node_modules/bootstrap/dist/css/'),
]);

una vez modificado la vista , debes bajar el servidor y levantarlo nuevamente para que tome los cambios hechos en app.js , para bajarlo debemos presionar las letras ctl+c

Si lo quisieras ocupar dentro de alguna vista ejs lo que debes llamar dentro de la etiqueta head

  <link rel="stylesheet" href="/assets/bootstrap.css">

luego invocar el script arriba del tag de termino de body

<script src="/assets/bootstrap.js"></script>
<script src="/actions/my_script.js"></script>
